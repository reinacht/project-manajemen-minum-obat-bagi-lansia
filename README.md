Aplikasi ini adalah perangkat lunak berbasis desktop yang dirancang untuk membantu pengasuh (caregiver) atau anggota keluarga dalam memantau jadwal minum obat lansia. Dengan antarmuka yang bersih dan mudah digunakan (user-friendly), program ini bertujuan meningkatkan kepatuhan minum obat melalui sistem pengingat otomatis yang dapat dipersonalisasi.

Fitur Unggulan
Manajemen Data Lansia & Obat:

Menyimpan profil lansia (Nama, Usia, Kondisi Kesehatan).

Mencatat detail obat secara rinci: Nama obat, dosis, jadwal waktu (bisa multiple), dan instruksi khusus (misal: diminum setelah makan).

Sistem Pengingat Cerdas (Smart Reminder):

Alarm real-time berupa pop-up window dan suara saat waktu minum obat tiba.

Fitur Snooze (Tunda) dan konfirmasi "Sudah Diminum".

Personalisasi Suara Alarm (Fitur Unik):

Memungkinkan pengguna mengunggah rekaman suara sendiri (format .wav/.mp3).

Contoh penggunaan: Menggunakan rekaman suara cucu atau anak untuk mengingatkan lansia, sehingga terasa lebih personal dan akrab.

Pencatatan Riwayat (Log History):

Merekam secara otomatis kapan obat diminum untuk memudahkan pemantauan kepatuhan harian.

Penyimpanan Otomatis:

Semua data (profil, obat, riwayat) tersimpan otomatis dalam format JSON (elderly_data.json), sehingga data tidak hilang saat aplikasi ditutup.

Spesifikasi Teknis
Bahasa Pemrograman: Python 3.

Framework GUI: Tkinter (Tema Elegant dengan palet warna soft).

Audio Engine: Pygame (dengan fallback ke Winsound jika Pygame tidak tersedia).

Penyimpanan Data: JSON (Local Storage).
